    生命游戏由剑桥大学数学家John Horton Conway在1970年提出来的。
    
    生命游戏构成：
    1）元胞分布在规则划分的二维网格上；
    2）元胞具有0,1两种状态，0代表‘死’，1代表‘生’；
    3）元胞以相邻的8个元胞为邻居。即Moore邻居形式；
    4）一个元胞的生命由其在该时刻本身的生死状态和周围邻居的状态决定。
    
    生命游戏的规则：
    · Survival(生存)：对一个活的元胞，如果它的邻居中有两个或三个元胞是活的，那么该元胞继续生存。
    · Die(死亡)： 对一个活的元胞，如果它的邻居有三个以上元胞是活的，那么该元胞将死去；如果它的邻居少于两个，那么该元胞死去。
    · Born(繁殖)：对于一个空元胞，如果它的邻居中有且仅有三个元胞，那么该元胞将成为一个活的元胞。
